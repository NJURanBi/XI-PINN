# XI-PINN

Physics-Informed Neural Networks (PINNs) have emerged as a powerful class of mesh-free methods for solving partial differential equations (PDEs), particularly those involving complex geometries. In this work, we present an innovative Extended Interface Physics-Informed Neural Network (XI-PINN) framework specifically designed to solve parabolic moving interface problems. The proposed approach adopts a level set function to characterize the interface. This level set function can be either prescribed analytically or learned via a neural network. We carry out a rigorous a priori error analysis for the XI-PINN method, deriving error bounds for the approximation. The versatility of XI-PINN is further highlighted by its application to the Oseen equations. We perform comprehensive numerical experiments to validate the efficacy, accuracy, and robustness of the proposed framework.
